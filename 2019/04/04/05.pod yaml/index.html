Pod Yaml

YAML 基础

它的基本语法规则如下：

  1. 大小写敏感
  2. 使用缩进表示层级关系
  3. 缩进时不允许使用Tab键，只允许使用空格。
  4. 缩进的空格数目不重要，只要相同层级的元素左侧对齐即可
  5. # 表示注释，从这个字符一直到行尾，都会被解析器忽略

  1. Lists
